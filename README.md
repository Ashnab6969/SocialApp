# Social Sparks – Android App

**Social Sparks** is a native Android application built with **Kotlin** that helps users quickly find social interaction suggestions based on the time of day. It’s designed to be fun, engaging, and easy to use.

---

## 📱 App Features

- **Time input**: Enter a time of day (e.g., “Morning”, “Afternoon Snack Time”, “Dinner”).
- **Social spark suggestion**: The app displays a friendly suggestion using an if/else (when) statement.
- **Reset button**: Clears the input field and the current suggestion.
- **Error handling**: Friendly error messages for empty or invalid inputs.
- **Logging**: Key actions are logged to Logcat for debugging.
